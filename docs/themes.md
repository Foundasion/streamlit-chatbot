# Streamlit Themes Collection

This document contains a collection of custom themes for the AI Hub Streamlit application.

## Dark Options

### 1. Midnight Indigo
A sophisticated dark theme with a deep gray background and rich indigo accents, perfect for modern applications.

```toml
[theme]
base="dark"
primaryColor="#6366F1"  # Rich indigo accent
backgroundColor="#1E1E1E"  # Deep gray background
secondaryBackgroundColor="#2D2D2D"  # Lighter gray sidebar
textColor="#FFFFFF"  # White text
font="sans serif"
```

### Features
- Modern tech-inspired color palette
- Rich indigo accents provide sophistication with energy
- Excellent contrast with white text
- Professional and easy on the eyes
